# 701Client 出勤報表轉換器 - 操作說明

本程式用於將 701Client 匯出的原始 Excel 考勤資料，轉換為格式化的月度出勤報表（Excel/PDF）。

## 🚀 快速開始 (Quick Start)

1. **開啟程式**：執行 `main.py` 或應用程式捷徑。
2. **匯入人員名單**（首次使用或名單變更時）：
   - 若右側「人員名單」為空，請至選單列 `檔案` -> `匯入人員名單` 選擇 CSV 檔。
   - 格式需求：CSV 檔需包含人員姓名與類別（內勤/外勤）。
3. **選擇來源檔案**：
   -點擊下方 **「選擇檔案」** 按鈕。
   - 選取由 701Client 匯出的原始 Excel 考勤檔（檔名通常包含日期，如 `MonRep250101...xlsx`）。
4. **設定轉換參數**（可選）：
   - 在左側確認 **上下班時間設定**（內勤/外勤）是否正確。
   - 設定 **出席率最低標準**（預設 80%）。
   - 勾選 **生成 PDF**（若需要）。
5. **匯入請假名單**（如有）：
   - 點擊 **「匯入請假名單」** 選擇請假記錄檔。
6. **開始轉換**：
   - 點擊綠色的 **「轉換」** 按鈕。
   - **遇到未知人員**：若來源檔中有不在名單上的人員，程式會跳出視窗詢問，請選擇該員為「內勤」或「外勤」，系統將自動更新名單並繼續轉換。
7. **查看結果**：
   - 轉換成功後會顯示檔案路徑，通常位於與來源檔同目錄下的 `output` 資料夾或您指定的路徑。

## ⚙️ 功能詳解

### 1. 介面配置

- **左側面板**：
  - **上下班時間**：可個別設定內/外勤的「上班」與「下班」緩衝區間。
  - **出席率設定**：設定門檻值，低於門檻顯示紅色，接近門檻顯示黃色，合格顯示綠色。
  - **輸出設定**：可自訂輸出路徑及檔名格式。
- **右側面板**：顯示目前載入的內/外勤人員名單統計。
- **選單列**：
  - **主題**：可切換深色 (Dark) 或淺色 (Classic White) 介面模式。
  - **幫助**：提供將來源檔中的所有姓名匯出為文字檔的功能。

### 2. 常見問題排除

- **無法寫入檔案**：請確認輸出的 Excel/PDF 檔案沒有被其他程式開啟。
- **找不到人員名單**：請確認 `內外勤人員名單.csv` 是否存在或路徑正確。
- **格式錯誤**：請確保來源 Excel 檔案格式未被手動修改，應保持 701Client 匯出的原始結構。
